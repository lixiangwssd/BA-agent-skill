## 三、根因分析

### 3.1 主要拖累：付费推广（贡献度 71.9%）

付费推广渠道 GMV 从 310 万元暴跌至 195 万元，单周下降 115 万元（-37.1%），是本次总 GMV 下滑的核心元凶，占全部跌幅的近 72%。

可能原因：

| 排查维度 | 典型假设 |
|----------|----------|
| 投放预算 | 上周是否削减了付费投放预算？ |
| 出价策略 | 竞价策略调整导致曝光量或点击率骤降？ |
| 素材/落地页 | 广告素材或落地页更换后转化率下降？ |
| 平台政策 | 搜索引擎/信息流平台算法变动、账户审核？ |
| 竞争环境 | 竞争对手加大投放，CPC 成本飙升，ROI 转差后系统自动降投？ |
| 季节/促销 | 前周是否有大促撬动了付费流量，上周恢复正常？ |

重点排查：
1. 查付费渠道后台：曝光量、点击量、点击率、转化率、CPC、ROI 周环比数据
2. 确认预算有无变化，是否触发智能投放的自动降速机制
3. 检查账户健康状态（违规警告、审核拦截等）

### 3.2 次要拖累：自然搜索（贡献度 25.0%）

自然搜索 GMV 下降 40 万元（-9.5%），贡献了约 1/4 的跌幅。

可能原因：核心关键词排名下滑、站内搜索算法更新、付费缩水后引流协同效应减弱、用户主动搜索意愿季节性下降。

### 3.3 次要拖累：社群裂变（贡献度 21.9%）

社群裂变 GMV 下降 35 万元（-38.9%），跌幅比例仅次于付费推广。

重点排查前周是否有裂变专题活动（红包、拼团）在上周自然到期，若属正常回落则无需过度干预；若非活动原因，需排查分享率与裂变链路健康度。

### 3.4 亮点：直播带货（+30 万元，+16.7%）

直播带货逆势增长，GMV 占比从 18.0% 跃升至 25.0%。建议确认是否因头部 KOL 带货所致，并评估能否持续复制，同时考虑将部分付费推广预算向直播方向倾斜。

---

## 四、总结与建议

**核心结论**：上周 GMV 下滑主要由付费推广大幅缩量驱动，建议运营和投放团队今日内完成账户诊断并恢复投放节奏，同时将直播带货作为近期重点增长抓手加以放大。

| 优先级 | 行动项 |
|--------|--------|
| P0（立即） | 排查付费推广账户状态、预算、出价、素材转化率 |
| P1（今日内） | 确认社群裂变下滑是活动到期还是运营问题，必要时启动新裂变活动 |
| P2（本周内） | 分析自然搜索流量，评估付费缩量的协同负效应，制定 SEO 补偿方案 |
| P3（持续跟踪） | 总结直播带货增长经验，评估扩大直播投入的可行性 |

---

## 关键发现摘要

- **最大拖累**：付费推广，下降 115 万元（-37.1%），贡献了总跌幅的 71.9%，属异常级别波动，必须优先排查
- **次要拖累**：自然搜索（-40 万元）和社群裂变（-35 万元），合计贡献约 47% 的跌幅
- **唯一亮点**：直播带货逆势增长 +30 万元（+16.7%），部分对冲了跌幅
