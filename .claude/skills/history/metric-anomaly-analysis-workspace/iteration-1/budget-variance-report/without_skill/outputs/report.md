## 三、根因分析

**华南区（-102 万元）— 一次性时间差异（Timing Difference）**
- 已知背景：大客户 3 月延期签单，属偶发事件而非结构性问题
- Q2 该订单落单后预计追赶性反弹

**华北区（-55 万元）— 结构性偏弱，根因待定**
五维排查清单：
1. 销售漏斗健康度（线索量、商机转化率）
2. 人员稳定性（是否有关键销售离职）
3. 竞争态势变化（竞品是否加大投入）
4. 大客户决策周期是否拉长
5. Q1 季节性叠加执行不足

**华东区（+56 万元）— 最佳实践，建议复制**
华东超额 7%，应萃取成功经验向华北推广。

**西部区（+21 万元）— 稳健超额**
持续超额完成，保持当前节奏。
