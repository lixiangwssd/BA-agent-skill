# CPA 上涨原因分析

## 概况

上周整体 CPA 从 42 元涨到 59 元，涨幅约 40%。主要原因是消耗增加（+23%）的同时新客数下降（-12%）。

## 各渠道分析

| 渠道 | CPA变化 | 消耗变化 | 新客变化 | 判断 |
|------|---------|---------|---------|------|
| 信息流 | 45→72(+60%) | +30% | -19% | 主要问题 |
| 搜索 | 40→40(不变) | -10% | -10% | 正常 |
| 短视频 | 35→50(+43%) | +47% | +3% | 效率低 |

## 原因分析

### 信息流（主因）
- 消耗大幅增加但新客反降，说明转化效率严重下降
- 结合已知背景：换了新素材，很可能是新素材效果不好
- 竞品大促也可能抢走了部分用户

### 短视频（次因）
- 消耗增长很快但新客几乎没增加
- 可能是新增预算打到了低效人群

## 建议

1. 信息流素材效果差的先暂停，切回旧素材
2. 短视频控制预算增速，ROI 达标再加量
3. 搜索渠道效率稳定，可以适当加量
4. 竞品大促期间避免硬碰硬
