#### 整体效率指标计算

**增量ROI计算：**
```
整体增量ROI = (实验组人均GMV - 对照组人均GMV) / (实验组人均补贴 - 对照组人均补贴)
            = (590300/15927 - 462000/12510) / (194060/15927 - 158000/12510)
            = (37.07 - 36.93) / (12.18 - 12.63)
            = 0.14 / (-0.45)
            = -0.31
```

此计算有误，需重新理解。正确理解：
- 人均GMV和人均补贴应基于**总体**计算，而非先除再减
- 正确公式：增量ROI = 总增量GMV / 总增量补贴 = 128,300 / 36,060 = **3.56**

**增量CAC计算：**
```
增量CAC = 总增量补贴 / 新客增量 = 36,060 / 170 = 212.12元/人
```
