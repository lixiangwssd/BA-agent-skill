# 美团外卖补贴策略优化报告

**生成时间**：2026-05-14
**数据来源**：模拟数据（日均预算1000万元已验证 ✓）

---

## 一、核心结论概述

**当前策略评估：钱效中等偏低，存在优化空间**

| 指标 | 数值 |
|------|------|
| 日均总预算 | **1000万元** ✓ |
| 日均总用户 | 490万 |
| 整体增量ROI | **3.13** |
| 整体补贴率 | 约7.1% |

**核心问题：**
1. 新客补贴率偏高（10-14%），增量ROI仅2.3，建议降至10%上限
2. 低效渠道（搜索引擎增量ROI 1.6、老客低活渠道1.8-2.0）占用预算
3. 高效渠道（订单页增量ROI 5.2、搜索推荐4.5）预算分配不足

**优化方向：**
- 新客补贴率从14%降至10%，节省**35万/天**
- 高效渠道（订单页+搜索推荐）增加30%，增加**61万/天**
- 低效渠道（老客低活）收缩20%，节省**21万/天**
- 早餐时段增加36%，增加**18万/天**

**预期效果：**
- 整体增量ROI：**3.13 → 3.50+**（提升约12%）
- 总预算：1000万 → 约995万（节省5万）
- GMV预计提升8-10%
